# Интерфейс Electron приложения

## Внешний вид

Приложение отображает современный интерфейс с следующими элементами:

### Основные компоненты:
- **Заголовок**: "Hello World!" крупным шрифтом
- **Сообщения**: 4 блока с текстом в стиле карточек
- **Подзаголовок**: "Добро пожаловать в мир десктопных приложений!"
- **Техническая информация**: "Создано с помощью Electron + JavaScript"

### Дизайн:
- **Фон**: Градиент от синего (#667eea) к фиолетовому (#764ba2)
- **Контейнер**: Полупрозрачный с эффектом размытия (backdrop-filter)
- **Карточки**: Полупрозрачные с зеленой левой границей
- **Типографика**: Современные системные шрифты
- **Эффекты**: Тени, скругленные углы, стеклянные эффекты

### Размеры окна:
- Ширина: 800px
- Высота: 600px
- Адаптивный дизайн

### Цветовая схема:
- Основной фон: Градиент
- Текст: Белый с тенями
- Карточки: Полупрозрачный белый
- Акценты: Зеленый (#4CAF50)
