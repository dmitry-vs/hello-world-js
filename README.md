# Hello World на JavaScript

Простой проект Hello World на JavaScript, который выводит текст в консоль.

## Запуск

```bash
# Установка зависимостей (если нужно)
npm install

# Запуск проекта
npm start
```

Или напрямую через Node.js:

```bash
node index.js
```

## Что делает

Проект выводит в консоль:

- "Hello World"
- "Привет, мир!"
- "Это простой проект на JavaScript"
- "Текст выводится в консоль с помощью console.log()"
