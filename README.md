# Hello World на JavaScript + Electron

Простой проект Hello World на JavaScript, который выводит текст в консоль и в графическое окно с помощью Electron.

## Возможности

- **Консольный режим**: Вывод текста в терминал
- **GUI режим**: Красивое окно с помощью Electron
- **Современный дизайн**: Градиентный фон и стеклянный эффект

## Запуск

### Консольный режим
```bash
# Установка зависимостей
npm install

# Запуск консольной версии
npm start
```

### GUI режим (Electron)
```bash
# Запуск Electron приложения
npm run electron
```

## Что делает

### Консольная версия
Выводит в консоль:
- "Hello World"
- "Привет, мир!"
- "Это простой проект на JavaScript"
- "Текст выводится в консоль с помощью console.log()"

### Electron версия
Отображает в окне:
- Красивый интерфейс с градиентным фоном
- Анимированные сообщения
- Современный дизайн с эффектами
- Тот же контент, но в графическом виде

## Структура проекта

- `index.js` - консольная версия
- `main.js` - основной файл Electron приложения
- `index.html` - интерфейс для Electron
- `package.json` - конфигурация проекта
